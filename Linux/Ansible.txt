> now create a simple playbook
    #vim webserver.yml [must have yml extension]
    ---
    - name: webserver
      hosts: webserver [mention specefic hosts group from inventory file]
      remote_user: root
      become: true
      tasks: 
        - name: install httpd
          yum:
            name: httpd
            state: latest
        - name: run httpd
          service:
            name: httpd
            state: started
        - name: create content
          copy:
            content: "Welcome the Ansible Lab"
            dest: /var/www/html/index.html
            
> now verify playbook
#ansible-playbook webserver.yml --syntax-check
 playbook: webserver [if it's all ok show playbook name]
 
 
> now deploy playbook
#ansible-playbook webserver.yml

How to Create Partition using ansible
- name: Create a new ext4 primary partition
    device: /dev/sdb
    number: 1
    state: present
    fs_type: ext4

- name: Remove partition number 1
    device: /dev/sdb
    number: 1
    state: absent

- name: Create a new primary partition with a size of 1GiB
    device: /dev/sdb
    number: 1
    state: present
    part_end: 1GiB

- name: Create a new primary partition for LVM
    device: /dev/sdb
    number: 2
    flags: [ lvm ]
    state: present
    part_start: 1GiB
